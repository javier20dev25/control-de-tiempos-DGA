# CorintoFlow Logistics Monitoring System

Premium web platform adapted for mobile devices to monitor container flow Lead Times at the Port of Corinto.

## Features
- **OCR Integration**: Pattern-based extraction of ISO 6346 container IDs.
- **Role-based Views**: Specialized interfaces for Gate 5, JH Entrance, and JH Exit.
- **Logistics Analytics**: Lead time, lead time breakdown, and bottleneck detection.
- **Premium Design**: Dark mode with glassmorphism and micro-animations.

## Tech Stack
- Vite + Vanilla JS
- Tesseract.js (OCR)
- Chart.js (Analytics)
- Lucide (Icons)

## Installation
```bash
npm install
npm run dev
```
